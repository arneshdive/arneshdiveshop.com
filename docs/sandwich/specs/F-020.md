# F-020: Promotions (promo code creation, management, application)

> **Module:** Promotions · **Priority:** 4/100 · **Depends on:** F-001, F-011 · **Source:** prd.md:131-134

Discount code creation/management by admin and application of a promo code to an order, including validity periods.

## Scope

**In:**
- Promo code creation and management (admin)
- Apply promo code to an order during checkout
- Promotion validity periods

**Out:**
- Automatic/rule-based promotions without a code

## Acceptance Criteria

- [ ] **AC1** — Admin can create a promo code with a discount value and validity period
- [ ] **AC2** — Customer can apply a valid promo code at checkout and see the discount reflected in the order total
- [ ] **AC3** — An expired or invalid promo code is rejected with a clear message
