### UF-003 — Customer places order with Midtrans payment
- **Actor:** Customer
- **Trigger:** Customer proceeds to checkout from cart
- **Confidence:** `[stated]`

**Steps:**
1. Customer reviews cart contents andProceeds to checkout
2. Customer enters or selects shipping address
3. System calculates shipping, tax, and applies any promo code
4. Customer confirms order and initiates payment
5. System creates order atomically (order + items + payment record)
6. System redirects to Midtrans Snap payment page
7. Customer completes payment via chosen method (GoPay, bank transfer, QRIS)
8. Midtrans sends webhook confirming payment
9. System updates order status to 'processing'
10. Customer sees order confirmation page

**Outcome:** Order placed and payment confirmed, customer receives confirmation
