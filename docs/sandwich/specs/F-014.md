# F-014: Admin order list & detail view

> **Module:** Order Management · **Priority:** 10/100 · **Depends on:** F-002, F-013 · **Source:** prd.md:99-100

Admin order list with filtering (status, date range, search) and order detail view with customer info, items, and payment.

## Scope

**In:**
- Order list with filtering by status, date range, and search
- Order detail view showing customer info, items, and payment
- Served via Route Handlers (HTTP) for the admin's TanStack Query layer

**Out:**
- Order status changes (separate feature)

## Acceptance Criteria

- [ ] **AC1** — Admin can filter the order list by status and date range
- [ ] **AC2** — Admin can search orders
- [ ] **AC3** — Admin can open an order and see customer info, line items, and payment details
