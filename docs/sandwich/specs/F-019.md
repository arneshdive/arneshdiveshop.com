# F-019: Admin dashboard (sales metrics, revenue chart, top-selling products)

> **Module:** Admin Dashboard · **Priority:** 5/100 · **Depends on:** F-002, F-005, F-013 · **Source:** prd.md:121-124

Dashboard with sales metrics and charts, revenue reporting by period, and top-selling products analytics.

## Scope

**In:**
- Stats cards: total sales, orders, customers, revenue
- Revenue chart by time period
- Top-selling products list
- Date range filter for analytics

**Out:**
- Exportable/scheduled reports beyond the CSV export in Order Management

## Acceptance Criteria

- [ ] **AC1** — Dashboard shows current totals for sales, orders, customers, and revenue
- [ ] **AC2** — Admin can view a revenue chart filtered by date range
- [ ] **AC3** — Admin can see the top-selling products for the selected period
