# F-003: Email/password authentication

> **Module:** Auth · **Priority:** 14/100 · **Depends on:** F-001, F-002 · **Source:** prd.md:auth-module

Credential-based auth with NextAuth.js v5, password hashing, session management, middleware protection, RBAC

## Scope

**In:**
- NextAuth.js v5 configuration
- Credentials provider setup
- Password hashing with bcrypt or similar
- Session management with JWT or database sessions
- Middleware for protected routes
- Role-based access control (customer, admin, super_admin)
- Login/register forms
- Auth state in React context

**Out:**
- Google OAuth
- OTP verification
- Password reset flow

- [ ] **AC1** — User can register with email and password
