# F-021: Homepage banner CRUD & display

> **Module:** Banners · **Priority:** 11/100 · **Depends on:** F-001, F-002 · **Source:** prd.md:141-143

Admin banner CRUD and active-banner display on the storefront homepage carousel.

## Scope

**In:**
- Banner CRUD for admin (image, link, display order, active/inactive)
- Active banner display on the storefront homepage carousel

**Out:**
- Per-user or geo-targeted banners

## Acceptance Criteria

- [ ] **AC1** — Admin can create, edit, delete, and reorder banners
- [ ] **AC2** — Admin can toggle a banner active/inactive
- [ ] **AC3** — The homepage carousel shows only active banners in the configured order
