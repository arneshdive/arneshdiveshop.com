# F-005: OTP verification via Mailgun

> **Module:** Auth · **Priority:** 11/100 · **Depends on:** F-003 · **Source:** prd.md:auth-module

OTP generation, email sending via Mailgun, verification flow with 15-minute expiry

## Scope

**In:**
- OTP code generation (6 digits)
- OTP storage with expiry (15 minutes)
- Email template for OTP
- Mailgun API integration
- OTP verification endpoint
- Rate limiting for OTP requests

**Out:**
- SMS OTP
- OTP for password reset

## Acceptance Criteria

- [ ] **AC1** — User receives OTP email within 60 seconds of registration
- [ ] **AC2** — OTP expires after 15 minutes
- [ ] **AC3** — User can verify OTP to complete registration
- [ ] **AC4** — Rate limiting prevents OTP spam (max 3 per 10 minutes)
