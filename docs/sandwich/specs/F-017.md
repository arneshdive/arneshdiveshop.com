# F-017: Customer account (profile, order history, saved addresses)

> **Module:** Customer Management · **Priority:** 10/100 · **Depends on:** F-002, F-013 · **Source:** prd.md:111-113

Customer-facing account pages: profile management, order history view, and saved addresses management.

## Scope

**In:**
- Customer profile view/edit
- Order history view for the logged-in customer
- Saved addresses management (add/edit/delete)

**Out:**
- Admin-side customer list (separate feature)

## Acceptance Criteria

- [ ] **AC1** — Customer can view and edit their profile
- [ ] **AC2** — Customer can see their own order history
- [ ] **AC3** — Customer can add, edit, and delete saved addresses
