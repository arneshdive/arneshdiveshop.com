# Arne's Dive Shop — Product Requirements Document

> Generated by sandwich/order · mode: answer · 2026-07-05

## Overview
E-commerce platform for a dive shop featuring ISR-powered storefront and dynamic admin dashboard. Built with Next.js 15, PostgreSQL, Drizzle ORM, and Midtrans payment integration. Deployed on Vercel with Mailgun for transactional emails and Vercel Blob for image storage. Emphasis on vertical slice architecture with clean separation of domain, application, and presentation layers.
